# CricScoreLive

CricScoreLive is a beginner-friendly cricket website that shows live cricket scores, IPL match updates, and cricket news.  
This website is made using basic web technologies and hosted for free on GitHub Pages.

## What this website offers
- Live cricket score (demo)
- IPL updates
- Cricket news
- Mobile responsive layout

## Built With
- HTML
- CSS
- JavaScript
